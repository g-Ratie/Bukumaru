# ぶく丸 - Pixiv小説ブックマークビューアー

ぶく丸は、Pixivの小説ブックマークを効率的に検索・閲覧できるWebアプリケーションです。Next.js 15のApp Routerを活用し、TypeScriptとTailwind CSSで構築されています。

## データ

いい感じにブックマークデータを持ってくる必要があります
pixivのAPIのレスポンス準拠なのでよしなにやってください

## 技術スタック

- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS v4
- **UI Components**: shadcn/ui (Radix UI primitives)
- **Code Quality**: Biome (linting & formatting)
- **Testing**: Vitest + Testing Library
- **Storybook**: Component development & documentation
- **Package Manager**: Bun
- **Git Hooks**: Lefthook

このリポジトリはhttps://github.com/imaimai17468/imaimai-front-templateをベースにしています。
